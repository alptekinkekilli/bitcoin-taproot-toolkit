Sparrow Wallet Import — Taproot Single-Sig

⬇ Sparrow butonuyla indirilen .descriptor dosyasındaki MASTER_TPRV satırını aşağıdaki adımlarla Sparrow'a yükle.

  1. Sparrow → File → New Wallet → cüzdana isim ver
  2. Script Type: Taproot (Single Sig)
  3. Keystore 1 → Master Private Key (BIP32) → Enter Private Key
  4. MASTER_TPRV: altındaki tprv8ZgxMBi... satırını yapıştır
  5. Derivation: m/86'/1'/0' — otomatik gelir, değiştirme
  6. Import Keystore → Apply

Sparrow Receive sekmesindeki adres = frontend'deki adres. Transaction'lar Bitcoin Core (testnet4) üzerinden otomatik görünür.
